# Weather Information Application

A secure web application that retrieves and displays weather information with authentication and authorization using Auth0.

- **Weather Dashboard**: Fetches and displays weather data for multiple cities
- **Responsive Design**: Optimized for both desktop and mobile devices

- **Authorization**: Protected routes requiring user authentication
- **Multi-Factor Authentication**: Email verification for enhanced security
- **Data Caching**: 5-minute cache expiration for weather data
- **RESTful API**: Integration with OpenWeatherMap API
